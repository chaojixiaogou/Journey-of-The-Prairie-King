# Journey of The Prairie King

一个基于 Unity 引擎开发的 2D 射击游戏项目，灵感来源于《星露谷物语》（Stardew Valley）中的街机小游戏“草原之王的旅程”（Journey of the Prairie King）。

## 📌 项目简介

本项目旨在复刻并扩展《星露谷物语》中的街机小游戏。玩家将扮演一名西部牛仔，在充满敌人的关卡中射击、躲避并最终击败 Boss，体验快节奏的横版射击乐趣。

## 🛠 技术栈

- **游戏引擎**: Unity
- **主要语言**: C# 
- **图形着色器**: ShaderLab 和 HLSL

## 📂 项目结构

```
Journey-of-The-Prairie-King/
├── Assets/           # 游戏资源文件（脚本、场景、素材等）
├── Packages/         # Unity 包管理配置
└── ProjectSettings/  # Unity 项目设置
```

## 🚀 功能特性

- 经典横版射击玩法
- 多样化的敌人类型
- Boss 战斗机制
- 粒子特效与视觉优化
- 可扩展的游戏架构

## 📥 如何运行

1. 克隆本仓库：
   ```bash
   git clone https://github.com/chaojixiaogou/Journey-of-The-Prairie-King.git
   ```

2. 使用 Unity Hub 打开项目文件夹

3. 在 Unity 编辑器中点击 Play 按钮即可开始游戏
